# Repo Notes

- This is a public GitHub repo. Do not add secrets, and avoid over-exposing sensitive local details.
- The repo exists to keep config files from your various machines in version control. When you reference files like `.zshrc`, prefer the repo copy instead of inspecting the local machine outside this repo unless you explicitly ask for that.
- `homedir/` is a replacement for `~`.
- Any other directory in this repo should mirror the exact path layout from the local machine.
